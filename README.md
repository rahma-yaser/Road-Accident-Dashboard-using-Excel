## Project Overview

This project is a **comprehensive Road Accident Data Analysis Dashboard developed using Microsoft Excel**. The main objective is to analyze road accident casualties and uncover patterns related to accident severity, vehicle type, road conditions, location, time, and environmental factors. The dashboard converts raw accident data into meaningful insights through structured data cleaning, analysis, and interactive visualizations.

The project demonstrates strong skills in **data analysis, Excel-based reporting, and dashboard design**, without using external BI tools.

---

## Data Cleaning & Preparation (Excel)

Before building the dashboard, the dataset was cleaned and prepared using Excel to ensure accuracy and consistency:

* Removal of duplicate and irrelevant records
* Handling missing and blank values
* Standardization of categorical fields (severity, road type, surface, location)
* Formatting date fields for time-based analysis
* Validation of numeric fields for aggregation
* Creation of helper columns to support pivot tables and KPIs

These steps ensured the dataset was analysis-ready and reliable.
